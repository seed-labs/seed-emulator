### 2. 🌐 虚拟企业网络环境
**需要接入的开源项目**:
- **网络拓扑**: GNS3 或 EVE-NG
- **虚拟机管理**: VirtualBox/VMware + Vagrant
- **容器化**: Docker + Docker Compose

**网络架构**:
```
DMZ区域:
├── Web服务器 (nginx + 漏洞应用)
├── 邮件服务器 (Postfix + 钓鱼检测)
└── DNS服务器 (BIND + DNS隧道检测)

内网区域:
├── 域控制器 (Samba4 AD)
├── 文件服务器 (SMB + 敏感文件)
├── 数据库服务器 (MySQL + 业务数据)
├── 工作站群 (Windows/Linux 客户端)
└── 打印机等IoT设备

管理网络:
├── 安全设备 (防火墙、IDS/IPS)
├── 日志服务器 (ELK Stack)
└── 监控中心 (Grafana + Prometheus)
```
