# Ready-to-Paste Figure Prompts

Use these directly with your image model. Both are intended for white-background, 16:9, proposal-grade figures.

## Prompt A: Architecture Figure

```text
Create a proposal-grade scientific architecture figure on a pure white background, 16:9, ultra-clean, macro, quiet, and highly polished. This is not a product UI, not a dashboard, and not an engineering workflow poster. It should look like a top-tier systems or security paper figure, suitable for a USENIX-style paper and a national science proposal.

Title: Trustworthy Agentic Orchestration for Closed-Loop Network Experimentation

The figure should communicate one central idea: a realistic network emulator becomes a disciplined scientific instrument only when three layers are integrated into a closed, auditable loop.

Layer 1 at the bottom: High-Fidelity Network Simulation Substrate. Show a calm but substantial network world with autonomous systems, routers, services, realistic runtime behavior, faults, and attack scenarios. It should feel executable and alive, not like a toy topology.

Layer 2 in the middle: Runtime Control and Evidence Plane. Show structured runtime inventory, selector-scoped operation, deterministic jobs, logging, measurement, packet capture, verification outputs, and durable artifacts. Emphasize observability, bounded intervention, and evidence persistence.

Layer 3 at the top: Trustworthy Agentic Orchestration. Show objective formalization, constraints, policy gates, human confirmation for elevated actions, reasoning over structured state, bounded planning, and evidence-aware summarization. It should feel strategic and methodological, not like a chat interface.

Across the whole figure, show one dominant closed loop: observe, plan, act, verify, archive. The loop should be large, elegant, and visually dominant, spanning all three layers. The visual hierarchy should be sparse and strong: three large conceptual regions, very few small boxes, short labels only.

Typography should be refined, black or dark charcoal. Palette should use restrained blue-gray structure, muted teal directional flow, and subtle warm mineral highlights for evidence and assurance. Use generous whitespace, soft geometry, minimal clutter, and precise alignment. Avoid any product flavor, terminal flavor, shell flavor, API flavor, or DevOps flavor.

The final image must feel authoritative, methodological, and paper-ready.
```

## Prompt B: Principle Figure

```text
Create a high-end scientific principle diagram on a pure white background, 16:9, elegant, sparse, and conceptually powerful. It should look like a strong principle figure from a top-tier systems, networking, or security paper. Do not draw a UI, dashboard, or engineering control panel.

Title: Methodological Principle of Trustworthy Autonomous Experimentation

The figure should explain one core principle: autonomous experimentation becomes trustworthy only when grounded observation, bounded intervention, supervised escalation, verification, and evidence archival are jointly enforced as one closed-loop discipline.

Center the composition around one large closed-loop cycle with five stages: Observe, Formulate, Intervene, Verify, Archive.

Around this main cycle, show four methodological constraints:
1. Grounded State: decisions rely on structured runtime inventory, routing state, service health, and measured evidence from running systems.
2. Bounded Action: interventions are limited by selector scope, deterministic execution surfaces, and explicit policy tiers.
3. Supervised Escalation: elevated actions cross a visible policy gate and, when required, a human confirmation gate.
4. Reproducible Evidence: every consequential step leaves artifacts, traces, and verification outputs that support replay, comparison, and audit.

The figure should suggest that these constraints transform a free-form LLM agent into a disciplined scientific orchestrator. Use concise intellectual labels such as objective, admissible action, policy gate, verification, evidence archive, reproducibility, and replayable trace.

Use refined geometry, soft blue-gray structural fields, muted teal directional flow, and subtle gold or mineral accents for assurance and evidence. Strong whitespace. Minimal clutter. No code windows, no terminals, no chat bubbles, no software chrome.

The finished image should feel like a mature methods figure for a proposal and a paper.
```
